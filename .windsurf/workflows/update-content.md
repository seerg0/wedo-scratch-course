# Обновление содержания сайта

Этот workflow помогает обновить главную страницу сайта (`index.html`) после добавления новых уроков.

## Когда использовать

- После создания нового урока
- При изменении порядка уроков
- При обновлении описания курса

## Шаги для обновления содержания

### 1. Откройте `index.html`
Найдите секцию навигации:
```html
<nav class="nav">
    <h2>📚 Навигация по курсу</h2>
    <ul>
        <!-- Сюда добавляются новые уроки -->
    </ul>
</nav>
```

### 2. Добавьте новый урок в навигацию

**Формат ссылки:**
```html
<li><a href="[Номер]_[название]_[тип].md">[Номер]. [Название] ([Тип])</a> - [краткое описание]</li>
```

**Пример:**
```html
<li><a href="3_машина_wedo.md">3. Машина (WeDo)</a> - создаем движущийся автомобиль</li>
```

### 3. Проверьте порядок уроков

Уроки должны идти в логическом порядке:
1. **Формат курса** (всегда первый)
2. **WeDo уроки** (нечётные номера)
3. **Scratch уроки** (чётные номера)
4. **Чередование:** WeDo → Scratch → WeDo → Scratch

### 4. Обновите статистику (если нужно)

Проверьте, нужно ли обновить:
- Общее количество уроков
- Описание курса
- Информацию о прогрессе

### 5. Проверьте HTML валидность

Убедитесь, что:
- Все ссылки закрыты `</a>`
- Все `<li>` закрыты `</li>`
- Нет синтаксических ошибок

### 6. Тестирование локально (опционально)

```bash
# Откройте index.html в браузере
open index.html
```

Проверьте:
- Все ссылки работают
- Навигация выглядит корректно
- Стили применяются правильно

### 7. Загрузите изменения

```bash
git add index.html
git commit -m "update: add navigation for lesson [Номер]"
git push origin main
```

## Пример полного обновления

**Было:**
```html
<li><a href="1_сигнализация_wedo.md">1. Сигнализация (WeDo)</a> - изучаем логику "ЕСЛИ → ТО"</li>
<li><a href="2_сигнализация_scratch.md">2. Сигнализация (Scratch)</a> - переносим логику из LEGO в игру</li>
```

**Стало после добавления урока 3:**
```html
<li><a href="1_сигнализация_wedo.md">1. Сигнализация (WeDo)</a> - изучаем логику "ЕСЛИ → ТО"</li>
<li><a href="2_сигнализация_scratch.md">2. Сигнализация (Scratch)</a> - переносим логику из LEGO в игру</li>
<li><a href="3_машина_wedo.md">3. Машина (WeDo)</a> - создаем движущийся автомобиль</li>
```

## Результат

Обновленная главная страница с новой навигацией, которая отражает все доступные уроки курса.
