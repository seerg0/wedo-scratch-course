# 🌀 Урок 5: Продвинутый умный вентилятор (Scratch)

## 🎯 Задача урока
Сделать «умный» вентилятор, который сам регулирует свою скорость в зависимости от температуры.

## 🧰 Подготовка

Создаём переменные:
- `температура`
- `скорость`

---

## 🧪 Практика

### 🔹 Шаг 1: Температура

```
когда нажат зелёный флаг
установить температура в 20
```

Добавим управление:

```
когда клавиша ↑ нажата
изменить температура на 1

когда клавиша ↓ нажата
изменить температура на -1
```

---

### 🔹 Шаг 2: Логика работы

ВАЖНО: в Scratch нет "иначе если", поэтому делаем вложенные условия

```
если <температура > 25> тогда
    установить скорость в 5
иначе
    если <температура > 20> тогда
        установить скорость в 3
    иначе
        установить скорость в 0
```

---

### 🔹 Шаг 3: Вращение вентилятора

```
всегда
    повернуть на (скорость) градусов
```

👉 Чем больше скорость — тем быстрее вращение!

---
